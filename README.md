# node_sysfs_gpio

A Node.js manager  for sysfs of gpio

## Installation

Install via git clone

```bash
$ git clone https://github.com/david6miji/node_sysfs_gpio.git
$ cd node_sysfs_gpio/app
$ npm install
```

## Run

run server

```bash
$ DEBUG=app:* npm start
```
